Pasta NCM2 - condições inicias no modelo CFD: direção NO, velocidade inicial 2m/s
Pasta NCM8 - condições inicias no modelo CFD: direção S, velocidade inicial 2m/s

Cada pasta contém 24 ficheiros, sendo que cada ficheiro apresenta dados para diferentes alturas (com  
uma resolução de 2m). 
O nome do ficheiro representa a altura do corte. 
O primeiro ficheiro apresenta dados para uma altura de 2m e o último ficheiro apresenta dados para 
uma altura de 48m.

Cada ficheiro é constituido por 4 colunas, x, y, vel, dir.
x,y representam as coordenadas do dominio do modelo CFD, com uma resolução de 2m, sendo que a primeira
coordernada corresponde ao canto inferior esquerdo do dominio de simulação. 
A coluna vel representa a magnitude da velocidade do vento, em m/s.
A coluna dir representa a direção do vento.
